# BSK-Stack: Хроматический трёхфазный стек с динамическим старением элементов

BSK-Stack представляет собой новую стековую структуру данных, основанную на принципе *хроматического распределения* элементов по трём динамическим зонам — White, Blue и Red. В отличие от классических стеков, данная модель вводит механизм возрастной стратификации, регулирующий доступность элементов в зависимости от времени их нахождения в стеке.

## Концепция и мотивация

BSK-Stack предназначен для задач, в которых требуется:

- гарантированное сохранение глубинных (исторических) данных;
- предотвращение удаления элементов старших уровней;
- моделирование процессов с «необратимым старением» состояний;
- управление приоритетом обработки событий по времени их появления.

Элементы распределяются по трём зонам:

- **White** — полностью доступные элементы;
- **Blue** — элементы со сниженным приоритетом, но извлекаемые;
- **Red** — защищённые элементы, недоступные для операции pop().

## Формальная модель

Пусть стек содержит `N` элементов. Границы зон вычисляются по параметрам `w` и `b`:

```
White zone: 0 ≤ i < floor(w * N)
Blue zone:  floor(w * N) ≤ i < floor((w + b) * N)
Red zone:   floor((w + b) * N) ≤ i < N
```

Операция `pop()` разрешена только для элементов зон White и Blue.  
Элементы Red-зоны могут быть извлечены только при уменьшении размера стека (их зона сдвигается вверх при изменении `N`).

## Научная новизна

BSK-Stack предлагает:

- формальную модель трёхфазного стекового пространства;
- механизм автоматического динамического старения элементов;
- строгие инварианты, исключающие доступ к защищённой зоне Red;
- обобщение LIFO-поведения с введением фазовых ограничений;
- возможность применения аналитических методов для изучения переходов между зонами.

Модель может рассматриваться как гибрид между стеком, приоритетной структурой и защищённым логом.

## Области применения

BSK-Stack подходит для:

- систем телеметрии и потоковой аналитики;
- защищённых логов и временных хранилищ;
- буферизации событий с ограничениями на удаление;
- симуляции процессов с фазовым старением объектов;
- многоуровневых стеков задач и событий.
