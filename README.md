\section*{BSK-Stack: Хроматический трёхфазный стек с динамическим старением элементов}

BSK-Stack представляет собой новую модель стековой структуры данных, основанную на принципе \textit{хроматического распределения} элементов по трём динамическим зонам: White, Blue и Red. В отличие от классических стеков, где все элементы обладают одинаковой доступностью, предлагаемая модель вводит механизм возрастной стратификации, обеспечивая дифференцированный доступ к данным в зависимости от их ``давности'' в стеке.

\subsection*{Концепция и мотивация}
Основной идеей является адаптация поведения стека к задачам, где требуется:
\begin{itemize}
    \item контролируемое ограничение на извлечение глубоких элементов;
    \item сохранение ``исторических'' данных в неизменяемой зоне;
    \item моделирование процессов с необратимым старением состояний;
    \item управление приоритетом событий по времени их появления.
\end{itemize}

Элемент, находящийся ближе к вершине стека, принадлежит зоне White и полностью доступен. Более глубокие элементы переходят в зоны Blue и Red, причём последняя является защищённой: элементы в ней недоступны для операции \texttt{pop()}. Границы зон определяются непрерывными параметрами $w$ и $b$ и пересчитываются при каждом изменении размера стека.

\subsection*{Формальная модель}
Пусть стек содержит $N$ элементов. Тогда зоны определяются выражениями:
\[
W = \lfloor wN \rfloor, \qquad B = \lfloor (w+b)N \rfloor,
\]
где $w, b \in [0,1]$ и $w + b \le 1$. Индексы элементов лежат в диапазонах:
\[
\begin{aligned}
&\text{White}: && 0 \le i < W, \\
&\text{Blue}:  && W \le i < B, \\
&\text{Red}:   && B \le i < N.
\end{aligned}
\]

Операция \texttt{pop()} разрешена только для элементов в первых двух зонах. Таким образом, BSK-Stack стабильно сохраняет содержимое ``глубокой'' части структуры, предотвращая его случайное или преждевременное удаление.

\subsection*{Цели и область применения}
Предложенная модель ориентирована на системы, где требуется одновременно:
\begin{itemize}
    \item приоритетная обработка новых данных;
    \item гарантированное сохранение исторической информации;
    \item моделирование многослойных буферов событий;
    \item контроль доступа по возрастному признаку.
\end{itemize}

Эта структура применима в задачах телеметрии, потоковой аналитики, ведения защищённых логов, а также при моделировании процессов с фазовым переходом или накоплением необратимых состояний.

\subsection*{Научная новизна}
BSK-Stack предлагает:
\begin{itemize}
    \item формальную модель трёхзонного стека с параметризацией зон;
    \item механизм динамического ``старения'' элементов в зависимости от их положения;
    \item строгие инварианты, исключающие повреждение или потерю глубоких данных;
    \item возможность математического анализа переходов между зонами;
    \item обобщение традиционного LIFO-поведения с фазовыми ограничениями.
\end{itemize}
